## 📝 使用指南

### 快速开始
```bash
# 安装依赖
yarn install

# 启动开发服务器
yarn dev

# 运行测试
yarn test:run

# 类型检查
yarn type-check

# 生产构建
yarn build
```

### 环境配置
```bash
# 复制环境变量模板
cp .env.example .env

# 配置后端 API 地址
VITE_API_BASE_URL=http://localhost:9001/api/v1
```
